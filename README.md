## 简介

误判统计小程序是一款基于 **Python + Tkinter** 开发的桌面应用程序，旨在帮助质检人员快速、高效地统计图片缺陷的误判率。通过简洁直观的界面和便捷的快捷键操作，大幅提升质检工作效率。

## 核心特性

### 图片管理
- 批量加载整个文件夹的图片
- 支持多种图片格式：JPG、PNG、BMP、GIF、TIFF
- 自动遍历子目录，一次性加载所有待检图片
- 支持前后翻页浏览，快速定位

### 图片查看
- **鼠标滚轮缩放**：无损缩放图片，细节清晰可见
- **鼠标拖拽移动**：自由移动图片位置，查看任意区域
- **适应窗口显示**：自动调整图片大小以适应显示区域

### 统计功能
- 自定义误判类型，灵活适配不同质检标准
- 实时计算并显示统计数据
- 支持总产能设置，准确计算误判率
- 分类统计各类型误判数量和占比
- 自动保存统计结果，随时导出报告

### 操作体验
- 工业风格UI设计，界面简洁专业
- 全屏显示模式，专注质检工作
- 快捷键支持，提升操作效率
- 支持重新标注，容错性强

## 使用指南

### 基本操作流程

```
选择文件夹 → 查看图片 → 选择判定结果 → 下一张 → 完成统计
```

### 1. 选择图片文件夹

点击菜单 **文件 → 选择文件夹**，选择包含待检图片的文件夹。

程序会自动：
- 扫描文件夹中的所有图片
- 显示图片总数
- 加载第一张图片

### 2. 图片查看操作

| 操作 | 功能 |
|------|------|
| 鼠标滚轮 | 放大/缩小图片 |
| 鼠标拖拽 | 移动图片位置 |
| ← → 按钮 | 前后翻页查看 |

### 3. 判定操作

对于当前显示的图片：

**检出（无缺陷）**
- 点击绿色 **"检出"** 按钮
- 或按快捷键 **D**

**误判（有缺陷）**
1. 勾选对应的误判类型（可多选）
2. 点击红色 **"误判"** 按钮
- 或按快捷键 **M**

判定后自动切换到下一张图片。

### 4. 统计数据说明

底部统计区实时显示：

- **总产能**：设置的总产能数量（默认为图片总数）
- **误判/检出数量**：已标注的图片统计
- **误判率** = 误判数量 / 总产能 × 100%
- **检出率** = 检出数量 / 总产能 × 100%
- **各类型误判率**：每种误判类型的占比
- **当前进度**：已标注数量 / 图片总数

### 5. 高级功能

**设置总产能**

点击菜单 **配置 → 设置总产能**，输入总产能数量。

默认情况下，程序会使用图片总数作为总产能。

**配置误判类型**

点击菜单 **配置 → 设置误判类型**，可以：
- 添加新的误判类型
- 删除现有类型
- 修改类型配置

配置会自动保存到 `config.json`。

**导出统计报告**

点击菜单 **文件 → 导出统计报告**，将统计数据导出为TXT格式文本文件。

**重新标注**

通过前后翻页按钮返回之前标注的图片，重新选择判定结果即可更新统计。

## 计算公式

```
误判率 = (误判数量 / 总产能) × 100%
检出率 = (检出数量 / 总产能) × 100%
类型误判率 = (该类型误判数量 / 总产能) × 100%
```

所有百分比保留两位小数。

## 开发与打包

### 打包成可执行文件

**Windows系统：**

```bash
# 直接运行打包脚本
build.bat
```

**Linux/Mac系统：**

```bash
# 给脚本添加执行权限
chmod +x build.sh

# 运行打包脚本
./build.sh
```

打包完成后，可执行文件位于 `dist/误判统计小程序.exe`。

### 手动打包

```bash
# 安装PyInstaller
pip install pyinstaller

# 执行打包命令
pyinstaller --onefile --windowed --name "误判统计小程序" main.py
```

## 常见问题

<details>
<summary><b>Q: 程序启动时提示缺少依赖？</b></summary>

A: 请确保已安装 Pillow 库：
```bash
pip install Pillow
```
或使用可执行文件版本，无需安装任何依赖。
</details>

<details>
<summary><b>Q: 图片加载失败或显示异常？</b></summary>

A: 请检查：
1. 图片文件是否损坏
2. 图片格式是否在支持列表中
3. 文件路径是否包含特殊字符
</details>

<details>
<summary><b>Q: 误判率计算不准确？</b></summary>

A: 请检查总产能设置：
- 默认使用图片总数作为总产能
- 如需使用实际产能，请通过菜单设置
- 误判率 = 误判数 / 总产能 × 100%
</details>

<details>
<summary><b>Q: 统计数据保存在哪里？</b></summary>

A: 保存在程序目录下的 `results.json` 文件中，每次完成统计会话时自动更新。
</details>

<details>
<summary><b>Q: 如何修改误判类型？</b></summary>

A: 通过菜单 **配置 → 设置误判类型** 进行添加或删除操作，至少需保留一个类型。
</details>

<details>
<summary><b>Q: 支持哪些图片格式？</b></summary>

A: 支持 .jpg、.jpeg、.png、.bmp、.gif、.tiff 等常见格式。
</details>

<details>
<summary><b>Q: 可以重新标注之前的图片吗？</b></summary>

A: 可以。使用左右箭头按钮返回之前的图片，重新选择判定结果即可。程序会自动更新统计数据。
</details>
